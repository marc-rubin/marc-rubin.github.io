The macro models can be found here.

There are three files in this folder:

  1. Automated Prototype Excel Model File (toolkit.xlsx). Look to the first tab Table of Contents In the Master Excel file for the nine templates (S.I thru S.IX) that cover the major analytical insights produced by applying the “Tool Kit.”
  2. Manual Version of Excel File used as the basis for coding up the automated version (master.xlsx). This file could be referenced for instructors who want to dive into the weeds in class and examine the methods for developing the templates (tool kit) from the ground up. This includes all data acquisition and model calculations.
  3. User’s Guide (Word Document) to accompany the Automated Toolkit (to be added). Look to the Table of Contents for the discussion of the specific models covered in the “Tool Kit.” For the most part, each theoretical item in the toolkit is standardized in terms of the relevant model goals, methodologies employed, interpretation on numerical model output, location of the predictor and outcome variables in the spreadsheets and the key working assumptions.
